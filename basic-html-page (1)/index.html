Basic HTML Page

This page demonstrates some of the tools used to create the structure of a page.

It's really important to use descriptive HTML tags because they improve accessiblity so that more people can use and enjoy you site. Easier to read HTML and improved SEO are two bonus benefits of using the proper tags.

Some of these tags include header, footer, nav, video, article, section, and main. I'll make a page later demonstrating more of these tags and leave a link here.

This is an H2 heading.

This is a paragraph. You could write lots of stuff in here or only a little bit if you don't have much to say. You could also just use lorem ipsum placeholder text.

Below is an image. It has an alt attribute that help with accessiblity because screen readers can use it and it will also display if the image fails to load. If the image is only used for decoration then the best practice is to use an empty alt attribute. I used the <figure> tag to hold the <image> and <figcaption> tags so I could get the caption below the picture.
